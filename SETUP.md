# ArtVerse - Setup & Launch Guide

## ✅ Project Status: COMPLETE

All components for ArtVerse full-stack Laravel application have been implemented and tested.

## 🚀 Quick Start (5 minutes)

```bash
# Navigate to project
cd d:\GitHub\ Repos\artverse

# 1. Ensure dependencies installed
composer install

# 2. Setup environment (one-time)
cp .env.example .env
php artisan key:generate

# 3. Fresh database with sample data
php artisan migrate:fresh --seed

# 4. Serve locally
php artisan serve
```

**URL**: http://localhost:8000

- `artworks` table with moderation fields
- `comments`, `ratings`, `likes` tables

## 🎨 Key Features

### Artist Workflow
1. Register as artist
2. Login to dashboard
3. Click "Upload New Artwork"
4. Fill details (title, description, image, category, tags)
5. Submit - Auto-moderation runs
6. View stats: likes, views, comments

### Visitor Workflow
1. Browse gallery at /gallery
2. Click artwork to view details
3. Like, rate, or comment
4. Visit artist profile
5. Explore more artworks

### Admin Workflow
1. Login with admin credentials
2. Go to /admin dashboard
3. See flagged artworks counter
4. Click "Review" on flagged items
5. Set status (approved/flagged/rejected)
6. Add authenticity label
7. Save decision

## 🛠️ Troubleshooting

**"Table not found"**
```bash
php artisan migrate:fresh --seed
```

**"Class not found"**
```bash
composer dump-autoload
```

**Port 8000 already in use**
```bash
php artisan serve --port=8001
```

**Permission denied**
```bash
chmod -R 775 storage bootstrap/cache
```
